# Cirrux CLI

Command-line interface for [Cirrux](https://cirrux.co).

## Install

```bash
brew install cirrux-co/tap/cirrux
```

## Usage

```bash
# Authenticate with your Cirrux account
cirrux login

# Show current user and workspace
cirrux whoami

# Log out of the current workspace
cirrux logout
```
